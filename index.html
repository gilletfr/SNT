L'activité

Créer un site web, c'est quoi ?

La SNT (Sciences Numériques et Technologie) est une matière obligatoire en classe de Seconde. Elle permet aux élèves de comprendre les technologies qui façonnent notre quotidien numérique.

L'un des thèmes clés du programme est le Web : son histoire, son fonctionnement, les protocoles HTTP, les langages HTML et CSS. La création d'un site personnel est l'activité pratique qui concrétise tous ces apprentissages.

Chaque groupe d'élèves a imaginé, structuré et codé un mini-site web en équipe, du choix du sujet jusqu'à la mise en ligne sur GitHub Pages.

<!-- Notre premier site web -->
<!DOCTYPE html>
<html lang="fr">
  <head>
    <title>Mon projet SNT</title>
  </head>
  <body>
    <h1>Bienvenue !</h1>
    <p>Fait avec ❤ en SNT</p>
  </body>
</html>
